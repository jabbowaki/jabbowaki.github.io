Looping in Ruby and Javascript Cheatsheet

RUBY!

For Loops

For loops are used to iterate through the code block and do one thing per expression. 


              Format:
              for variable in [variable]
                code block that will do something
              end
              Example:
              for x in 0..10
                puts "This number is #{i}"
              end

While Loops

While loops run the code blocks while a specified condition is true. 


              Format:
              while condition is equal to true do
                code block that will do something
              end
              Example:
              while x < 10 do
              puts x
              x +=1

Until

Until will run a code block while a specified condition is false. 


              Format:
              until code is true do
                code block that will do something
              end
              Example:
              until x > 10 do
                puts x
                x+=1
              end

JAVASCRIPT!

For Loops

For loops will iterate through a block a code a number of time


            Format:
            for(thing executed before loop starts; determines how long loop runs for; what's executed after each loop){
              code block that will do something
          }
            Example:
            var x = 0
            for (x = 0; x < 10; x++){
            console.log("this is number " + x +".")
            };

For/In Loops

A for/in loop will iterate through the properties of an object


          Format:
          for(var x in object){
            code block that does something
        }
          Example:
          for (var x in votes) {
            voteCount.president[votes[x].president] += 1
          };

While Loops

A while loop will loop through a block of code while a specific condition is true.


            Format:
            while(condition){
            code block that will do something
            }
            Example:
            var x = 0
            while (x<10){
            console.log("The number is " + x)
            x++;
            }

Do/While Loops

A Do/While loop iterates through a block of code while a specified condition is true.


            Format:
            do {code block that will do something
            } while (thing is true);
            Example:
            var x = 0
            do {
              console.log("The number is " + x)
              x++
            } while (x <10);
