Looping in Ruby and Javascript Cheatsheet

RUBY!

For Loops

For loops are used to iterate through the code block and do one thing per expression.

Example:
for x in 0..10
puts "This number is #{i}"
end

While Loops

While loops run the code blocks while a specified condition is true.

Example:
while x < 10 do
puts x
x +=1
end


Until

Until will run a code block while a specified condition is false.

Example:
until x > 10 do
puts x
x +=1
end

JAVASCRIPT!

For Loops

For loops will iterate through a block a code a number of time.

Example:
var x = 0
for(x=0; x<10; x++){
console.log("This is number" + x +".")
};

While Loops

A while loop will loop through a block of code while a specific condition is true.

Example:
var x = 0
while (x<10){
console.log("This is number" + x +".")
x++;
};

For/In Loops

A for/in loop will iterate through the properties of an object.

Example:
for(var x in votes)
voteCount.president[votes[x].president]

Do/While Loops

A Do/While loop iterates through a block of code while a specified condition is true.

Example:
do {
console.log("This is number" + x +".")
x++;
};
