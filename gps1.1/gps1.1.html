Pairing is Great!
By Baron and Emily

Importance of Pairing

Pairing is great for many reasons. Working together allows you to bounce ideas off of each other and check each other's work. It helps you to get comfotable talking about code and best practices. It is particularly important while working on a big project in which many people are involved so that consistancy can be maintained.

Driver Vs Navigator

Navigator provides overall direction for where the program should go and produce it's overall vision. The driver should be more hands on with the actual construction of the code and syntax so that you don't have both people producting conflicting code and style.

Reflections
